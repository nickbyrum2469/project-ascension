# Project Ascension — Consolidated Production Pass

This milestone is a broad implementation and verification pass built on the reconstructed Floor One foundation.

## In scope

- Persistent Foundation beacons used as checkpoints and respawn anchors.
- Exploration caches, Riftglass shards, restorative tonics, and weapon attunement progression.
- Additional city residents and simple ambient routines so Caelus Reach reads as inhabited.
- Additional frontier encounter groups using tuned variants of the existing creature framework.
- A northern Floor Guardian arena and persistent ascension gate progression.
- Clearer in-world navigation through beacon lights, route markers, and objective-aware HUD messaging.
- Browser-level runtime testing that starts the production build, enters the world, exercises both perspectives and combat inputs, fails on console/page errors, and uploads screenshots.
- Performance and quality gates before merge.

## Deferred production work

- Final externally-authored character meshes and motion-captured animation sets.
- Full authoritative multiplayer and account services.
- Complete voice-over coverage and localization.
- Additional floors beyond the first ascension threshold.
